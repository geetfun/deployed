# Deployed

Deployed is a web interface for the deployment library, [Kamal](https://kamal-deploy.org).

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "deployed"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install deployed
```
